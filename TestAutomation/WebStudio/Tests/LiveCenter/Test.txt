*** Test Cases ***

Create an entry and submit it
	Set Selenium Timeout 	15 seconds
	Close All Browsers
	Login To Escenic Live Center
	Open an Event
	Create Entry  First test entry of Escenic event
	Make Entry Sticky
	Select Author  elive	
	Submit Entry
	Page Should Contain Element  xpath=//div/p[contains(text(),'First test entry of Escenic event')]
        #Page Should Contain Element  xpath=//div/p[text()=' First test entry of Escenic event']

	#publish entry
	Publish the Submitted Entry  First test entry of Escenic event

	#create milestone entry 
	Create Entry  This is a milestone entry
	Click Element  css=input[class="ng-pristine ng-untouched ng-valid ng-scope"]
	Publish Entry

	Page Should Contain Element  xpath=//*[contains(text(), 'This is a milestone entry')]/../../../../../../..//label[contains(text(),'Milestone')]
